# Matrix Angular Project

Este proyecto genera matrices dinámicas en Angular con celdas de tamaño fijo y color uniforme, utilizando estilos de PrimeNG.

## Características
- Generación de matrices de 3x3, 4x4, 5x5 y 6x6.
- Uso de standalone components en Angular.
- Estilización con PrimeNG.
